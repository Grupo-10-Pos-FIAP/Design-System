# Spinner

Componente de loading/spinner para indicar processos em andamento.

## Visão Geral


O componente Spinner é usado para fornecer feedback visual durante operações assíncronas, como carregamento de dados, processamento de formulários ou qualquer ação que requeira tempo de espera.
